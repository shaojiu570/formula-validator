# ⚡ 快速开始

## 🎯 三步开始使用

### 第一步：安装依赖
```bash
npm install
```

### 第二步：启动项目
```bash
npm run dev
```

### 第三步：打开浏览器
自动打开 http://localhost:3000

---

## 📦 构建部署

### 构建生产版本
```bash
npm run build
```

### 三种使用方式

#### 1️⃣ 直接打开（最简单）
双击打开 `dist/index.html` 文件

#### 2️⃣ 本地预览
```bash
npm run preview
```
访问 http://localhost:4173

#### 3️⃣ 部署到服务器
将 `dist/` 目录上传到任何静态服务器
